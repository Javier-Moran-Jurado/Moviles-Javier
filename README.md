# Taller 1

## Descripción breve del taller
Este taller consiste en desarrollar una aplicación móvil en Flutter que demuestra el manejo de estado con StatefulWidget y setState(), integrando imágenes desde internet y assets locales, implementando widgets de diseño como Container, ListView, Row y Column, y utilizando SnackBar para retroalimentación al usuario. La aplicación muestra el nombre del estudiante, un banner de videojuegos, una lista de personajes favoritos y un botón que cambia dinámicamente el título de la AppBar.

## Pasos para ejecutar

1. Clonar el repositorio:

git clone https://github.com/Javier-Moran-Jurado/Moviles-Javier.git
cd taller1_flutter

2. Instalar dependencias:

flutter pub get

3. Conectar dispositivo Android (USB con depuración activada) o abrir emulador

4. Ejecutar la aplicación:
flutter run


5. Para recargar cambios durante el desarrollo, presionar r en la terminal donde corre flutter run.
